## Custom Font Assets

Drop the official DM Sans and Bebas Neue `.ttf` files in this directory using the naming convention below. The Expo font loader is already wired up to look for these exact filenames:

- `DMSans-100Thin.ttf`
- `DMSans-100Thin_Italic.ttf`
- `DMSans-200ExtraLight.ttf`
- `DMSans-200ExtraLight_Italic.ttf`
- `DMSans-300Light.ttf`
- `DMSans-300Light_Italic.ttf`
- `DMSans-400Regular.ttf`
- `DMSans-400Regular_Italic.ttf`
- `DMSans-500Medium.ttf`
- `DMSans-500Medium_Italic.ttf`
- `DMSans-600SemiBold.ttf`
- `DMSans-600SemiBold_Italic.ttf`
- `DMSans-700Bold.ttf`
- `DMSans-700Bold_Italic.ttf`
- `DMSans-800ExtraBold.ttf`
- `DMSans-800ExtraBold_Italic.ttf`
- `DMSans-900Black.ttf`
- `DMSans-900Black_Italic.ttf`
- `BebasNeue-Regular.ttf`

> Tip: these files are already included for convenience, but you can replace any weight with your own updated font file as long as you keep the filename identical.
