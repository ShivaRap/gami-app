# Lottie Animations

Place your Lottie JSON animation files here:

- `slow-pace-walk.json` - Animation for the Slow Pace walking screen
- `fast-pace-walk.json` - Animation for the Fast Pace walking screen

The animations will automatically:
- Auto-play when the screen loads
- Loop continuously
- Pause when the session is paused (speed = 0)
- Resume when the session resumes (speed = 1)
- Have a gradient overlay that transitions smoothly into the background color
